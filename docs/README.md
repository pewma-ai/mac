# Management as Code

> Gestión basada en documentos, procesos y decisiones explícitas.
> Para personas, equipos y organizaciones que quieran hacer gestión como se hace código: versionada, trazable y colaborativa — con humanos y agentes IA.

---

## ¿Qué es MaC?

MaC es una metodología de gestión que reemplaza reuniones informales y decisiones perdidas en chat por **documentos vivos** y **procesos explícitos**. Todo queda escrito, todo se puede revisar, todo tiene historia.

No importa si eres freelance o diriges un equipo de 80 personas. MaC escala porque su unidad básica es un archivo `.md`, no una herramienta.
