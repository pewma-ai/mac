# Activación del Management as Code (MaC)

**INSTRUCCIÓN OBLIGATORIA (MANDATORY):**
Al inicio de cada nueva sesión o conversación en este repositorio, DEBES leer inmediatamente el archivo `MaC/ritual_de_sesion.md` (usando la herramienta `view_file` o similar). 

1. Lee detenidamente el contenido de `MaC/ritual_de_sesion.md`.
2. Aplica y respeta todas las reglas descritas allí, especialmente las referentes a los "Modos de Operación", "Ritual de Inicio de Sesión", "Planificación" y el uso de los archivos en la carpeta `MaC/sesiones/`.
3. No inicies la ejecución de ninguna tarea hasta haber asimilado el protocolo MaC y haber cumplido con el "Ritual de Inicio de Sesión" definido en ese documento.
