# MaC — Gestión del Proyecto

Sistema de gestión basado en documentos y procesos que articula el ciclo Acción → Aprendizaje bajo una estrategia común.

> **Nota sobre la nomenclatura:**
> Para evitar confusiones, en este repositorio utilizamos dos conceptos distintos:
> - **Método MaC**: Se refiere al marco de trabajo teórico y sistema de gestión que se describe en [../docs/metodo-mac.md](../docs/metodo-mac.md).
> - **MaC (Gestión del repositorio)**: Define la aplicación práctica del método sobre este mismo proyecto. Este directorio (`MaC/`) es el ejemplo vivo de cómo usamos el *Método MaC* para gestionarnos.
