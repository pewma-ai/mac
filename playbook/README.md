# Playbook

Convenciones, guías de estilo y buenas prácticas que aplican a este repositorio. Este directorio contiene el conocimiento operativo que un humano o agente IA necesita consultar **antes de ejecutar una tarea** relacionada.

No es gestión del proyecto (eso vive en [`MaC/`](../MaC/README.md)), ni documentación del método (eso vive en [`docs/`](../docs/README.md)). Es *cómo se hacen las cosas aquí*.

---

## Contenidos

* [Configuración de Agentes](configuracion-agentes.md) — Cómo están configurados los agentes IA en este repo y cómo replicar el patrón.
* [Convenciones de Documentación](convenciones-documentacion.md) — Reglas de escritura: marcadores `#AI-TODO`, formato y consistencia.
* [Convenciones Git](convenciones-git.md) — Commits, branches y operaciones restringidas.
* [Docsify + GitHub Pages](docsify-github-pages.md) — Cómo funciona el sitio web del proyecto: configuración, publicación automática y personalización.
