# Management as Code (MaC)

MaC es un sistema de gestión potenciado por inteligencia artificial. Se basa en documentos markdown para responder preguntas clásicas de gestión, hace explícitos procesos de gestión con sus cadencias y rituales, y ayuda a definir y evolucionar las políticas internas que hacen única la organización.
